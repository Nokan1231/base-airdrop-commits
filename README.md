# TaskFlow - Simple Todo CLI Application

A lightweight command-line task management application built with Node.js.

## Features

- ✅ Add new tasks
- - 📋 View all tasks with status
  - - ✔️ Mark tasks as complete
    - - 🗑️ Delete completed tasks
      - - 💾 Persistent storage using JSON
       
        - ## Installation
       
        - ```bash
          git clone https://github.com/Nokan1231/base-airdrop-commits.git
          cd base-airdrop-commits
          npm install
          ```

          ## Usage

          ```bash
          node app.js add "My new task"
          node app.js list
          node app.js complete 1
          node app.js delete 1
          ```

          ## Project Structure

          - `app.js` - Main application file
          - - `tasks.json` - Task storage
            - - `package.json` - Project dependencies
